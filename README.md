## 📢 시스템 공지사항 관리 방법

본 시스템 상단에 노출되는 업데이트 소식은 저장소 내의 `notice.txt` 파일을 통해 실시간으로 변경됩니다.

### 1. 수정 방법
`notice.txt` 파일을 열고 아래 형식을 지켜서 한 줄로 작성하세요.
`[업데이트 날짜 및 제목] | [연결할 링크]`

### 2. 작성 예시
2026.01.14 이미지 드래그 이동 및 우클릭 삭제 기능 업데이트 | https://github.com/YourID/YourRepo#20260114-업데이트-상세

### 3. 주의사항
- 제목과 링크 사이에는 반드시 세로 바(`|`)를 넣어야 합니다.
- 파일의 첫 번째 줄만 인식하여 상단 바에 노출됩니다.
- 링크가 없거나 형식이 틀리면 상단 공지 바가 나타나지 않습니다.

---

### 🚀 2026.01.14 업데이트 상세 (고객사 확인용)
- **이미지 자유 이동:** 이제 사진을 마우스로 꾹 눌러 원하는 위치(텍스트 사이 등)로 드래그하여 옮길 수 있습니다.
- **삭제 방식 변경:** 실수로 클릭하여 사진이 지워지는 일을 방지하기 위해, **마우스 오른쪽 버튼(우클릭)**을 눌렀을 때만 삭제 메뉴가 나타납니다.
- **수정 모드 보호:** '수정' 버튼을 누르기 전에는 이미지 조작이 불가능하도록 안전장치가 추가되었습니다.
